# Rat Land
> title is a work in progress

**Rat Land** is a 2D metroidvania-style game built in **Godot**. 

## About the Project
- Developed with **Godot Engine** (GDScript)  
- Features:
  - Player movement and combat mechanics
  - Enemy AI
  - Multiple worlds with interactive elements
  - Original sprites and animations
- Iterative testing and refinement of gameplay loops
